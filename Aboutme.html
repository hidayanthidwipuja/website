Pendidikan

Saya adalah mahasiswa di Universitas Singaperbangsa Karawang Jurusan Informatika

Minat

Saya memiliki ketertarikan yang mendalam dalam pengembangan web dan teknologi digital. Saya senang mempelajari tren terbaru dalam desain UI/UX dan membangun aplikasi yang responsif serta user-friendly. Saya percaya bahwa sebuah website atau aplikasi yang baik tidak hanya berfungsi dengan baik, tetapi juga memberikan pengalaman yang nyaman bagi penggunanya. Oleh karena itu, saya selalu berusaha memperdalam pemahaman saya tentang desain interaksi, aksesibilitas, serta optimasi performa website agar lebih cepat dan efisien. Selain itu, saya juga tertarik untuk mengembangkan aplikasi berbasis web yang dapat menyelesaikan permasalahan sehari-hari, baik dalam dunia bisnis, pendidikan, maupun hiburan. Saya senang bereksperimen dengan berbagai teknologi seperti HTML, CSS, JavaScript, serta framework modern seperti React dan Laravel untuk menciptakan solusi digital yang inovatif. Dengan semangat belajar yang tinggi, saya terus mengikuti perkembangan teknologi terbaru agar dapat menciptakan aplikasi yang tidak hanya fungsional, tetapi juga mengikuti standar industri dan kebutuhan pengguna saat ini

Pengalaman

Saya memiliki pengalaman dalam mengembangkan aplikasi berbasis web menggunakan Laravel dan React. Dalam perjalanan ini, saya telah membangun berbagai fitur interaktif yang membantu meningkatkan efisiensi dan pengalaman pengguna. Saya memahami pentingnya backend yang kuat dan frontend yang responsif, sehingga saya selalu memastikan integrasi antara kedua aspek tersebut berjalan dengan optimal. Selain itu, saya juga memiliki pengalaman dalam mengelola database menggunakan MySQL dan Firebase, serta mengoptimalkan performa aplikasi agar lebih cepat dan scalable." Saya juga telah bekerja sama dalam tim untuk membangun aplikasi manajemen data untuk organisasi kampus. Dalam proyek ini, saya berperan dalam merancang arsitektur sistem, mengimplementasikan API, serta memastikan data dapat diakses dengan mudah dan aman oleh pengguna. Kolaborasi dalam tim menjadi pengalaman berharga bagi saya, di mana saya belajar bagaimana bekerja secara efektif dengan developer lain, memahami kebutuhan klien, serta menerapkan metodologi Agile dalam pengembangan perangkat lunak. Dengan pengalaman ini, saya semakin percaya diri dalam mengembangkan aplikasi yang dapat memberikan dampak nyata bagi pengguna dan organisasi.
