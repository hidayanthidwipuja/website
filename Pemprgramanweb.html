Materi Pembelajaran

Mata kuliah Pemrograman Web pada semester 4 program studi Informatika adalah salah satu mata kuliah inti yang berfokus pada pengembangan aplikasi berbasis web. Dalam mata kuliah ini, mahasiswa akan mempelajari berbagai konsep, teknologi, dan teknik pemrograman untuk membangun website yang dinamis, interaktif, dan terstruktur. Website yang baik tidak hanya tentang tampilan (frontend) tetapi juga tentang bagaimana data diproses dan dikelola di server (backend). Oleh karena itu, mahasiswa akan dikenalkan dengan berbagai teknologi yang mendukung pengembangan web, mulai dari HTML, CSS, dan JavaScript untuk tampilan dan interaksi, hingga PHP dan MySQL untuk mengelola data di sisi server.
